# Python Data Entry Automation

## 📌 Project Overview
This project automates product registration in a web system using Python.  
Data is read from a CSV file and automatically inserted into the system interface.

## 🎯 Objective
Simulate a real-world scenario where bulk product data needs to be inserted into a system efficiently, reducing manual work and human error.

## 🛠️ Technologies Used
- Python
- Pandas
- PyAutoGUI
- CSV file handling

## 🗄️ Database-Related Concepts Applied
- Data extraction from structured files
- Data iteration and processing
- Null value handling
- Automation of data entry processes

## 🚀 Possible Improvements
- Replace UI automation with direct database insertion
- Connect to PostgreSQL using psycopg2
- Implement logging and error handling

## 📚 Learning Focus
This project demonstrates Python as a support tool for data operations and automation processes related to database environments.
